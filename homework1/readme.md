# go语言实现selpg

## 功能介绍
根据所输入的参数和文件名，显示出相应的内容，[具体内容](https://www.ibm.com/developerworks/cn/linux/shell/clutil/index.html)。

## 参数介绍
-s [number] 开始页码
-e [number] 结束页码
-l [number] 指定行数
< [filename] 指定文件
